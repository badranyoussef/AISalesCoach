Design-noter

Bredde: 380px fast. Placering: centreret på skærmen (WindowStartupLocation = CenterScreen).

Baggrund: #0f0f15 — matcht til overlay-designet (A/B/C). Farveskema er konsistent på tværs.

Accent: Indigo (#6366F1) i stedet for blå (#4f9cf9 fra POC) — Design A bruger blå, B/C bruger indigo. Login kan matche det valgte overlay-design.

States: Normal → Loading (felter disabled, knap dæmpet) → Fejl (røde kanter + fejlbesked) → Success (luk vindue, åbn overlay).

POC-reference: Eksisterende LoginWindow.axaml er 360×300px — dette design er 380×320px (lidt mere luft).
